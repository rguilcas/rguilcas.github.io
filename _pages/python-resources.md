Welcome to the **Scientific Python for Geosciences** resource hub. This page centralizes tools, tutorials, and best practices for using Python in Earth system sciences.

## Core Libraries

- **[Xarray](https://xarray.pydata.org/)**: Essential for handling multi-dimensional labelled arrays (NetCDF, GRIB).
- **[Dask](https://dask.org/)**: Parallel computing for analytics at scale.
- **[Cartopy](https://scitools.org.uk/cartopy/)**: The standard for geospatial data processing and map projection.
- **[Pandas](https://pandas.pydata.org/)**: High-performance, easy-to-use data structures for tabular data.

## Tutorials & Guides

- **[Project Pythia](https://projectpythia.org/)**: A community-driven educational resource for the Geosciences.
- **[Research Software Engineering with Python](https://swcarpentry.github.io/python-novice-inflammation/)**: Software Carpentry's foundational lessons.
